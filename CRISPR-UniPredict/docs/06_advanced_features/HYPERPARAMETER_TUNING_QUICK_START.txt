COMMAND-LINE OPTIONS
====================

--n_trials N            Number of trials (default: 50)
--study_name NAME       Study name (default: crispr_optimization)
--storage URL           Storage URL (e.g., sqlite:///optuna_study.db)
--output_dir PATH       Output directory (default: results/hyperparameter_tuning)
--config PATH           Base config (default: configs/model_config.yaml)
--resume                Resume previous study


COMMON COMMANDS
===============

# Quick test (10 trials, ~2-3 hours)
python scripts/hyperparameter_tuning.py --n_trials 10

# Standard optimization (50 trials, ~3-5 days)
python scripts/hyperparameter_tuning.py --n_trials 50

# Extended optimization (100 trials, ~1-2 weeks)
python scripts/hyperparameter_tuning.py --n_trials 100

# With persistent storage
python scripts/hyperparameter_tuning.py \
  --n_trials 50 \
  --storage sqlite:///optuna_study.db

# Resume previous study
python scripts/hyperparameter_tuning.py \
  --n_trials 50 \
  --storage sqlite:///optuna_study.db \
  --resume

# Custom output directory
python scripts/hyperparameter_tuning.py \
  --n_trials 50 \
  --output_dir results/my_optimization


OPTIMIZATION STRATEGY
====================

Objective: Maximize Combined Score = (SCC + AUROC) / 2

Algorithm: TPE Sampler (Bayesian Optimization)
  • Efficient exploration-exploitation
  • Good for high-dimensional spaces
  • Learns from previous trials

Pruning: Median Pruner
  • Stops unpromising trials early
  • Saves computation time
  • Configurable thresholds

TROUBLESHOOTING
===============

"CUDA out of memory"
  → Reduce batch size range in script
  → Reduce number of epochs per trial

"No improvement after many trials"
  → Expand search ranges
  → Increase number of trials
  → Check if baseline is already optimal

"Trials are very slow"
  → Reduce epochs per trial
  → Use smaller dataset for tuning
  → Enable GPU acceleration

"Study not found"
  → Check storage URL
  → Start new study with different name
