================================================================================
                    TROUBLESHOOTING
================================================================================

"pip: command not found"
  Solution: Use python -m pip install -r requirements_complete.txt

"No module named torch"
  Solution: Reinstall PyTorch with GPU support
  pip install torch==2.0.0 --index-url https://download.pytorch.org/whl/cu118

CUDA not available (shows False)
  Solution: Check NVIDIA drivers with nvidia-smi
  Reinstall PyTorch with CUDA support

Permission denied
  Solution: Use virtual environment or add --user flag
  pip install --user -r requirements_complete.txt

Conflicting dependencies
  Solution: Clear pip cache
  pip cache purge
  pip install -r requirements_complete.txt --force-reinstall

Slow installation
  Solution: Use legacy resolver
  pip install -r requirements_complete.txt --use-deprecated=legacy-resolver

  ✓ Native CUDA support

  ✓ Requires index-url for CUDA
  ✓ Simpler setup
  ✓ Recommended for CPU/Development
