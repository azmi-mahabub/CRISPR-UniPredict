================================================================================
                        TROUBLESHOOTING
================================================================================

Q: "conda: command not found"
A: Install Miniconda from https://docs.conda.io/projects/miniconda/

Q: CUDA shows False
A: Check NVIDIA drivers: nvidia-smi
   Install from: https://www.nvidia.com/Download/driverDetails.aspx

Q: Slow installation
A: Use mamba (faster):
   conda install -c conda-forge mamba
   mamba env create -f environment.yml

Q: Package conflicts
A: Clear cache:
   conda clean --all
   conda env create -f environment.yml

Q: Permission denied
A: Run terminal as Administrator or use:
   conda env create -f environment.yml --force
